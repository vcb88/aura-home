# AURA: 100 Сценариев Использования в Умном Доме

Этот документ описывает 100 потенциальных сценариев взаимодействия пользователя с системой AURA, демонстрируя как прямые запросы, так и проактивное поведение интеллектуального агента.

**Предполагаемый набор устройств в умном доме:**
*   Датчики температуры и влажности (в каждой комнате)
*   Датчики открытия окон и дверей
*   Датчики присутствия (mmWave)
*   Датчики освещенности
*   Датчики протечки воды
*   Мониторы чистоты воздуха (CO2, VOCs, PM2.5)
*   Умные розетки с измерителями потребляемой мощности (для большинства электроприборов)
*   BLE-сканеры (для персонализированного определения присутствия)
*   Беспроводные умные выключатели/кнопки
*   Квартирные счетчики воды и электричества
*   Умный входной замок
*   Умные шторы/жалюзи
*   Умные камеры с распознаванием объектов (VLM)
*   Умный термостат/климат-контроль
*   Умные колонки (Алиса)
*   Умный телевизор/медиацентр
*   Робот-пылесос
*   Умный холодильник
*   Носимые устройства (фитнес-трекеры, умные часы)
*   Датчики шума

22. AURA: "Я вижу, вы проснулись. Открыть шторы?" (Датчик присутствия, умные шторы)
23. AURA: "Я вижу, вы смотрите телевизор. Приглушить свет?" (Умный ТВ, датчик освещенности)
24. AURA: "Я вижу, вы заснули на диване. Выключить телевизор и приглушить свет?" (Датчик присутствия, умный ТВ, свет)
